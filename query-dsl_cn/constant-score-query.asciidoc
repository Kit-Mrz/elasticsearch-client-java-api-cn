==== Constant Score Query

参见 https://www.elastic.co/guide/en/elasticsearch/reference/5.2/query-dsl-constant-score-query.html[Constant Score Query]

[source,java]
--------------------------------------------------
QueryBuilder qb = constantScoreQuery(
        termQuery("name","kimchy")      <1>
    )
    .boost(2.0f);                       <2>
--------------------------------------------------
<1> 查询
<2> 查询分数
