== Java管理API

Elasticsearch 提供了一套完整的 Java API 来处理管理任务.

要想访问他们, 你需要在客户端对象上调用 `admin()` 方法来获取一个 `AdminClient` 对象:

[source,java]
--------------------------------------------------
AdminClient adminClient = client.admin();
--------------------------------------------------

[NOTE]
在本指南的剩余部分, 我们将使用 `client.admin()`.
